## Features

&nbsp;&nbsp;**Chromecast** — Cast photos and videos to any Cast-compatible screen

&nbsp;&nbsp;**App Lock** — Biometric, face unlock or device PIN on every launch

&nbsp;&nbsp;**Multi-language** — English, Spanish, French, Portuguese, German + System default

&nbsp;&nbsp;**Hide media** — Filter out photos or videos from the gallery view

<br/>

---

<br/>

## Requirements

- Android **8.0 Oreo** or higher (SDK 26+)
- Storage permission (photos & videos)
